# generator-vue
Generador de componentes Vue


Ejecutar

<code>npm link</code>

Usarlo en Yeoman con

<code>yo vue Component</code>

o

<code>yo vue Component path/to/component</code>


Esto creara un directorio "Component" o "path/to/component/Component" con 4 archivos

+ Component.vue
+ Component.html
+ Component.js
+ Component.css
